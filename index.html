Git

Qu'est ce que Git ?

Git est un logiciel de versioning qui permet de garder une trace des modifications successives faites dans son code.

Comment utiliser Git ?

Avant toute chose, il faut télécharger Git sur : Git-scm.com
Prenez soi de choisir la version correspondant à votre système d'exploitation.

Installer Git... ;)

Ouvrez un terminal, dans le repertoire que vous souhaitez versionner

  1. git init
  2. git add --all
  3. git commit -m "nom du commit"

Vous avez commité, BRAVO !!!
